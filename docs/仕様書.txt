契約書・NDAチェッカー 仕様書


アプリ名
  日本語: 契約書・NDAチェッカー
  英語: Contract Checker
  フォルダ名: contract-checker


概要
  契約書やNDA (秘密保持契約) のテキストを貼り付けると、
  AIがリスク条項、不利な条件、注意すべきポイントをハイライトして解説するツール。
  法的な専門知識がなくても、契約書のリスクを大まかに把握できる。


対象ユーザー
  フリーランス、個人事業主、スタートアップ経営者、契約書を初めて確認する人。


注意事項
  本ツールは法的助言を提供するものではない。
  AIの分析結果は参考情報であり、重要な契約は必ず弁護士に相談すること。
  この注意書きをアプリ内に常時表示する。


主な機能

  契約書テキスト入力
    契約書の全文をテキストエリアに貼り付ける。
    契約種別の選択 (業務委託/NDA/売買契約/雇用契約/ライセンス契約/その他)。
    自分の立場の選択 (発注側/受注側/甲/乙)。

  AIリスク分析
    Gemini APIを使い、以下の観点で契約書を分析する。
      リスク条項: 一方的に不利な条件、過大な責任負担
      競業避止: 不当に広い競業禁止条項
      損害賠償: 上限のない損害賠償条項
      解約条件: 不均衡な解約条件
      知的財産: 成果物の権利帰属が不明確な箇所
      秘密保持: 秘密保持の範囲と期間
    各リスク項目にリスクレベル (高/中/低) を付与。
    条項ごとに平易な日本語で解説と改善提案を出力。

  チェック結果表示
    リスクのある箇所を色付きハイライトで可視化。
      高リスク: 赤
      中リスク: 黄
      低リスク: 緑
    サマリーとして全体のリスクスコアを表示。
    各項目をクリックで詳細解説を展開。

  履歴管理
    過去にチェックした契約書の一覧をLocalStorageに保存。
    契約書名、チェック日時、リスクスコアで一覧表示。


画面構成
  ホーム画面: 過去のチェック一覧、新規チェックボタン
  入力画面: テキスト入力エリア、契約種別と立場の選択
  結果画面: ハイライト付き契約文、リスクサマリー、各条項の詳細解説
  履歴詳細画面: 過去のチェック結果の再閲覧


技術スタック
  HTML / CSS / JavaScript
  Gemini API (契約書分析)
  LocalStorage (データ永続化)


データ保存
  LocalStorageに以下を保存する。
    契約書テキスト
    契約種別、立場
    分析結果 (リスク項目、スコア、解説)
    チェック日時
    APIキー


テーマ
